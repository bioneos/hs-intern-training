BioNeos Intern Software Setup

By the end of this session, your computer and accounts should be ready for Arduino, web development, containers, code editing, Slack, and GitHub.

Good Troubleshooting is Part of the Internship.

When something doesn't work, don't give up or ask for help too early. For the installs ask yourself:

  • Do you have the right installation copy? (Windows, x64, etc.)
  • Have you restarted after installs?
  • Is you school email blocking sign ups?
  • Are you using the right terminal?
  • Do you have any typos in your commands?

Try to troubleshoot, but it is okay to ask for help once you have tried a few different things.

Setup Mission

Today is not about memorizing every tool. It is about getting a working development environment and knowing how to verify it.

  1. Create the accounts you will use for collaboration.
  2. Install the software used in the intern workflow.
  3. Run one quick test for each install.
  4. Ask for help early when a step does not match your screen.

Steps

  1. Accounts: Slack + GitHub
  2. Editor: Visual Studio Code
  3. Web tools: Node.js
  4. Containers: Docker
  5. Hardware: Arduino IDE + ESP8266 drivers

Node.js + npm

Node.js lets your computer run JavaScript outside the browser. npm installs JavaScript packages such as Express.

Download Node.js

Node Check

Open a terminal and run:

node -v
npm -v
If both commands print version numbers, Node.js and npm are working.

Docker

Docker runs apps in containers so projects behave more consistently across different computers.

Download Docker Desktop

Docker Check

Open a terminal and run:

docker --version
If the command print a version number, Docker is installed.

Arduino IDE

Arduino IDE is used to write, compile, and upload code to microcontroller boards such as ESP8266 development boards.

Download Arduino IDE

ESP-12 / ESP8266

ESP8266 boards often need two setup pieces: Arduino board support and the USB-to-serial driver for the board's adapter chip.

Add ESP8266 Board Support

  1. Open Arduino IDE.
  2. Go to File > Preferences.
  3. Paste this URL in Additional Boards Manager URLs:
https://arduino.esp8266.com/stable/package_esp8266com_index.json
  1. Go to Tools > Board > Boards Manager.
  2. Search esp8266 and install the ESP8266 platform.
